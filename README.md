# Projetos Full Stack

Este repositório reúne alguns dos meus **projetos pessoais de desenvolvimento Full Stack**, criados durante meus estudos em programação.

Os projetos aqui presentes envolvem a construção de aplicações completas, utilizando diferentes tecnologias tanto no **front-end** quanto no **back-end**, com foco em prática e aprendizado.

## Tecnologias utilizadas

- HTML
- CSS
- JavaScript
- APIs REST
- Banco de Dados SQL

## Estrutura do repositório

Cada pasta contém um projeto independente com sua própria lógica e funcionalidades.

Exemplos de projetos presentes neste repositório:

- Buscador de CEP com integração de API
- Sistema simples de boletim escolar
- Aplicações web para prática de lógica e manipulação de dados

## Objetivo

Praticar e consolidar conhecimentos em:

- Desenvolvimento **Front-end**
- Desenvolvimento **Back-end**
- Integração com **APIs**
- Manipulação de dados
- Construção de interfaces web
- Estruturação de aplicações completas
